Bosch Lib

  • © 2016 – 2017

  • GitHub

A collection of Python libraries used for developing automated testing and QA for Bosch Security System panels.

  • mode2/ - collection of classes used to communicate with the panel using the proprietary Mode2 protocol.
  • lookup.py - collection of lookup dicts that were created by parsing specifications
  • pif.py - class for encapsulating an entire PIF binary dump, including parsing an XML specification of data offsets within the bin dump.
  • rpsaccount.py - class whose instances represent RPS accounts and their associated metadata.
  • util.py - utility methods for convertying bytes to hex strings.

PNC ZenPortal Requests

  • © 2015

  • Demo
  • GitHub

Requests are submitted while requesters and assignees are kept track of. A particular request goes through a predefined workflow with specific steps. During each step of the process, the requester and assignee are emailed about updates to a specific request. This application includes role-based permissions, including an administrative view that has global read and write permissions. This was developed using the FuelPHP framework.

This web application was developed for internal request tracking at PNC Bank as part of their IT tools and services.

© PNC Financial Services Group Inc. 2017
All rights reserved

TimeSys Buildtracker

  • © 2014 – 2015

An extension for the open-source Buildbot software. Written mostly in Python 2.7 with supporting scripts in Bash and Perl.

Like Buildbot, the extension uses SQLAlchemy for database ORM and Jinja2 for templating.

  • root.py - Primary python controller.
  • buildtracker_macros.html - Jinja2 template macros.
  • add_rdeps.pl - Perl script to select a random number of reverse dependencies.
  • file-build-failure.pl - Perl script to post a bug via XML-RPC.
© TimeSys LLC 2014
All rights reserved
