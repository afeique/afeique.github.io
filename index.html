Projects & Demos

A non-exhaustive list of things I've made, including re-purposed demonstrations of my professional work.

Maintenance Log

  • 2017
  • Demo
  • GitHub

A simple system that allows for the definion of a manufacturing or production workflow. This workflow can be arbitrarily long. Tickets can be submitted to the system and assigned to a particular step in a production workflow. Written using the Yii Framework.

This project was being developed at Bausch & Lomb as part of the production floor's IT and ticket management solution.

HtmlReports

  • 2016
  • GitHub

A set of Python 2.7 scripts designed to take JSON input and produce clean, navigable HTML reports. The HTML templates are written using Jinja2. Originally, Mako was going to be used as the templating engine.

This was developed at Bosch Security Systems as a plug-in reporter for an existing LabVIEW tool.

ZenPortal Requests

  • 2015
  • Demo
  • GitHub

Requests are submitted while requesters and assignees are kept track of. A particular request goes through a predefined workflow with specific steps. During each step of the process, the requester and assignee are emailed about updates to a specific request. This application includes role-based permissions, including an administrative view that has global read and write permissions. This was developed using the FuelPHP framework.

This web application was developed for internal request tracking at PNC Bank as part of their IT tools and services.

Bootman

  • 2015

Command-line perl script designed to manage kernel images for boot-testing boards connected to board-farm server. Complemented by a web-interface written using web.py.

  • main.pl - perl script used to manage boards.
  • setup.sh - bash script used to setup kernel images.
  • verify.pl - perl script used to aid in taking physical inventory and committing a list of valid boards to the database.
  • web.zip - web.py interface.

Buildtracker

  • 2014

An extension for the open-source Buildbot software. Written mostly in Python 2.7 with supporting scripts in Bash and Perl.

Like Buildbot, the extension uses SQLAlchemy for database ORM and Jinja2 for templating.

  • root.py - Primary python controller.
  • buildtracker_macros.html - Jinja2 template macros.
  • add_rdeps.pl - Perl script to select a random number of reverse dependencies.
  • file-build-failure.pl - Perl script to post a bug via XML-RPC.
Condenser Microphone
  • 2014

Attempted to design a condenser microphone for a capstone in Sensor System Design, Spring 2014. The final report closely follows IEEE format and is available in LaTeX.
